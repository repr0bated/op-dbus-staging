## Rollback Strategy

### Quick Rollback
```
sudo systemctl stop op-dbus-chat
sudo systemctl disable op-dbus-chat
```

### BTRFS Snapshot Rollback
```
# Create snapshot before deployment
sudo btrfs subvolume snapshot /opt/op-dbus/chat /opt/op-dbus/chat.backup

# Rollback
sudo btrfs subvolume delete /opt/op-dbus/chat
sudo btrfs subvolume snapshot /opt/op-dbus/chat.backup /opt/op-dbus/chat
```

### Overlayfs Rollback
```
# Unmount overlayfs
sudo umount /opt/op-dbus/isolated

# Remove upper layer (changes)
sudo rm -rf /opt/op-dbus/isolated.upper

# Remount (clean state)
sudo mount -t overlay overlay ... /opt/op-dbus/isolated
```
